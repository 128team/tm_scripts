# TM Scripts

A collection of userscripts by 128team.

## Scripts

---

### SG Toolkit Pro

Advanced toolkit for [SteamGifts](https://www.steamgifts.com/) — filters, inline enter, Steam ratings, sorting and more.

**[Install](https://raw.githubusercontent.com/128team/tm_scripts/main/SG_Toolkit/sg_toolkit.js)**

**Features:**
- Filter giveaways by level, entries, points, win chance, type (group / whitelist)
- Hide already entered giveaways
- Inline Enter / Leave buttons — no page reload needed
- Steam review ratings fetched directly from Steam API
- Sort by entries, chance, level, or cost
- Win chance display per giveaway
- Wishlist highlight glow
- Quick link to Steam Store page

---

### YummyAnime Grid View

Replaces the default anime list on the [YummyAnime](https://yummyanime.com/) profile page with a responsive grid layout featuring large posters.

**[Install](https://raw.githubusercontent.com/128team/tm_scripts/main/yummy-grid/yummy-grid.user.js)**
